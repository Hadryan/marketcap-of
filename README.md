## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/emilneander/marketcap-of.git
   ```
2. Install NPM packages in your folder
   ```sh
   npm install
   ```
3. Run project
   ```sh
   npm start
   ```
